# 마늘
학생들의 챗봇, 마늘

## 마늘이란?
마늘이란 이름은 딱히 의미가 있진 않고, 제가 고기를 먹을 때 같이 쌈싸먹고 구워먹는 마늘이 생각나서 그렇게 지었습니다.

## 마늘의 출시일은?
마늘의 현재 버전은 1.0이며, 2021년 1월 1일에 출시되었습니다.
마늘 Beta의 경우 1.1 베타테스트 중이며 상시 모집 중입니다.

## 마늘의 기능은?
마늘의 기능은 현재 지속적으로 추가 중입니다. 현재는 급식 불러오기, 코로나 19 정보 불러오기가 있습니다.
또, 자가진단을 자동으로 할 수 있습니다. 학교 정보를 볼 수도 있고 시간표 정보도 조회할 수 있습니다.

## 마늘의 예상 사용자는?
서일중학교의 학생입니다.
(급식: 서일중학교만 지원 / 코로나 19: 국내, 충남 현황만 지원 / 자동 자가진단: 서일중학교만 지원)

## 개발자는 누구?
현재 서일중학교에 재학 중인 학생입니다.

## 지원 기간은?
서일중학교를 졸업하는 동시(2023년 2월)에 종료되나, 진학할 학교의 챗봇으로 리뉴얼되어 서비스됩니다. 단, 사용자(서일중학교 재학 중인 사용자)가 50명 이상일 경우 1년 추가 연장되어 서비스됩니다.

### 추가 연장이란?
진학할 학교의 챗봇을 기본적으로 사용하나 명령어에 '서일중학교'라는 태그를 붙여 서일중학교 학생이 사용할 수 있게 할 생각입니다.

## 문제가 생기면?
챗봇의 상담직원 연결 기능을 사용하거나, 이 곳(Github)에 이슈를 남겨주시면 됩니다.

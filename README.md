# 마늘
학생들의 챗봇, 마늘

## 마늘이란?
마늘이란 이름은 딱히 의미가 있진 않고, 제가 고기를 먹을 때 같이 쌈싸먹고 구워먹는 마늘이 생각나서 그렇게 지었습니다.

## 마늘의 출시일은?
마늘 Beta는 2020년 11월 14일에 출시되었습니다.
마늘은 2021년 1월 1일에 출시되었습니다.
마늘 Beta의 경우 1.1 베타테스트 중이며 상시 모집 중입니다.

## 마늘의 기능은?

### 급식 조회 기능
마늘에서 서일중학교의 급식(중식, 석식)을 조회할 수 있습니다.
서일고등학교와 급식을 공유하기 때문에 고등학교 학생도 중식, 석식을 조회할 수 있습니다.(조식 지원 안 함)
마늘에서 가장 처음 구현한 기능입니다.

### 코로나 19 현황 조회 기능
마늘에서 코로나 19 현황(국내, 충남)을 조회할 수 있습니다.

| 내용 | 국내 | 충남 |
| :---: | :---: | :---: |
| 누적 확진자 | 지원 함 | 지원 함 |
| 치료 중 | 지원 함 | 지원 함 |
| 격리 중 | 지원 안 함 | 지원 함 |
| 검사 중 | 지원 함 | 지원 안 함 |
| 격리 해제 | 지원 함 | 지원 함 |
| 음성 결과 | 지원 함 | 지원 안 함 |
| 사망 환자 | 지원 함 | 지원 함 |

### 자동 자가진단 기능
마늘에서 서일중학교 학생들이 손쉽게 학생 건강상태 자가진단을 할 수 있습니다.
단, 2005년생부터 2009년생까지이며 서일중학교 재학생이어야 할 수 있습니다.
개인정보를 수집하거나 저장, 이용하지 않습니다.

### 시간표 조회 기능
마늘에서 서일중학교 시간표를 조회할 수 있습니다.

| 지원하는 반 |
| :---: |
| 1-1 |
| 1-2 |
| 1-3 |
| 2-1 |
| 2-2 |
| 2-3 |
| 3-1 |
| 3-2 |

### QR코드 생성 기능
마늘에서 QR코드를 생성할 수 있습니다.

## 마늘의 예상 사용자는?
서일중학교의 학생입니다.

## 개발자는 누구?
현재 서일중학교에 재학 중인 학생입니다.

## 지원 기간은?
서일중학교를 졸업하는 동시(2023년 2월 28일)에 종료되나, 진학할 학교의 챗봇으로 리뉴얼되어 서비스됩니다. 단, 사용자(서일중학교 재학 중인 사용자)가 50명 이상일 경우 1년 추가 연장되어 서비스됩니다.

### 추가 연장이란?
진학할 학교의 챗봇을 기본적으로 사용하나 명령어에 '중'이라는 태그를 붙여 서일중학교 학생이 사용할 수 있게 할 생각입니다.

## 문제가 생기면?
챗봇의 상담직원 연결 기능을 사용하거나, 이 곳(Github)에 이슈를 남겨주시면 됩니다.
